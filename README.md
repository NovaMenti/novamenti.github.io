**Prerequisites:**  Node.js

**Run locally:**
1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

**Deployment:**
Use **GitHub Pages + GitHub Actions** (recommended):
1. Push your code to `main`.
2. In GitHub: **Settings -> Pages -> Source: GitHub Actions**.
3. The action builds and deploys automatically on push.

Manual check (optional):
1. Build locally:
   `npm run build`
2. This creates the static output in `dist/`.
